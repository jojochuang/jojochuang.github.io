教學現場應用指南：數位工具如何支援中文聽讀寫

在海外或雙語環境教授中文時，老師常面臨教材版本多樣、學生背景差異大、課堂時間有限等挑戰。傳統紙本聽寫與字卡活動仍然重要，但若能在不重複製作大量投影片的前提下，讓「聽音辨義」與「聽寫書寫」快速對齊當週詞彙，就能把更多精力放在口語互動與閱讀策略上。本站連結的兩項工具，核心概念都是：由老師掌握試算表內容，再由瀏覽器即時載入，避免將學生姓名、成績等敏感資料上傳至第三方平台。

聽音辨字：從「辨音」到「辨義」的鷹架

初階學習者需要清楚的聲音線索與視覺提示；進階學習者則可能要減少注音或圖片提示，避免過度依賴。聽音辨字遊戲支援多種顯示模式，老師可依單元目標切換「圖片＋注音」「全注音」或「全中文」，並可搭配試算表中的英文提示欄位，協助只熟悉英文字母拼讀的學生建立語意連結。建議在課前先用示範連結試玩一輪，確認音檔或圖片網址在教室網路環境下可正常載入，再將自製試算表產生的專屬網址分享給學生或家長。

聽寫語詞：投影、計時與紙本的一致性

聽寫活動成功的關鍵之一，是讓學生清楚知道「要寫什麼」與「有多少時間可以思考」。聽寫語詞考試頁面支援全螢幕投影、每題計時與定時復述，並可將同一套詞表輸出為 PDF，方便印發給需要紙本作答的班級。老師可以依學習階段，選擇只顯示注音、只顯示漢字，或兩者混合，以符合課綱字音字形的重點。若同一校內有多位老師使用不同課本，建議每位老師各自複製範本試算表並設定共用權限，再將專屬網址儲存為書籤，即可避免內容彼此覆蓋。

試算表備課與版本控管習慣

Google 試算表適合協作與版本紀錄：老師可在「檔案 → 版本紀錄」中檢視先前內容，也可在課後微調詞表而不必重新製作整份活動頁。若學校政策不允許使用雲端試算表，仍可將資料匯出為 CSV，在支援本機載入的工具頁面使用（請參考各工具頁面的說明）。無論採用哪一種來源，都建議在課堂前用學生裝置實測一次，並預留離線備援活動，以降低網路不穩對教學節奏的影響。

與行政、資安政策的對話要點

資訊組或行政單位常關心：學生資料是否外流、是否符合 FERPA／各地個資法、以及第三方服務是否合規。本站首頁提供隱私權摘要，並另設獨立之 隱私權政策 頁面，方便學校審核。實務上，老師仍應避免在試算表公開欄位填寫可識別個資的內容，並依校方規範決定是否啟用額外分析或追蹤元件。若您需要書面說明，可將政策頁面網址一併提供給承辦人參考。

家長溝通與課後延伸

家長常詢問：孩子在家是否可以練習、以及是否需要註冊帳號。老師可說明：專屬連結多半只載入老師試算表中的公開練習內容，學生無須在本站建立帳號。課後延伸方面，可鼓勵學生聆聽同一語詞的課堂錄音或教科書附帶音檔，再對照試算表中的詞彙表自我檢核。對於學習速度較慢的學生，亦可將聽寫 PDF 列印為週複習單，降低螢幕使用時間。

總結而言，這些工具的定位是「減少重複勞動、提高練習次數」，而非取代教學設計。當試算表內容與學習單元緊密對齊、課堂規範清楚告知學生與家長時，數位聽讀寫活動更能穩定融入日常教學。若您有實際使用經驗或希望新增功能，也歡迎寄信與維護者討論，讓工具能持續反映第一線教室的需求。

LearnChinese: A Practical Guide for Teachers (English)

JOJOCHUANG · Maintainer of the LearnChinese portal · reynard99999@gmail.com

The LearnChinese site at jojochuang.github.io is a small, static gateway for educators who teach Chinese in bilingual schools, weekend heritage programs, or overseas community classrooms. It does not replace a full learning management system. Instead, it highlights two focused, browser-based utilities that many teachers can adopt quickly: a listening-and-choice activity and a timed dictation layout that exports to PDF. Both tools assume that the instructor already curates vocabulary in a spreadsheet, which matches how teams often share weekly word lists today.

Why spreadsheets remain the “source of truth”

Districts differ widely in approved software. Some classrooms rely on Google Workspace, while others can only open exported CSV files offline. The listening game and dictation pages therefore treat the teacher-authored sheet as the curriculum record. Columns can store Zhuyin-friendly text, image links, English glosses, and lesson numbers. When a teacher clicks the “custom content” buttons on this portal, the page only transforms a pasted Google Sheets URL into a parameterized link; it does not ask students to create accounts on this domain. That design choice reduces accidental collection of roster data and keeps the compliance story simple for administrators who review third-party tools.

Listening practice that scales with proficiency

Early learners usually need both audio and visible scaffolding. Intermediate learners may need to hide Chinese characters temporarily so they rely on Zhuyin or tone perception. Advanced groups might prefer an all-Chinese card layout with minimal hints. The linked listening game exposes those display modes through URL parameters, which means teachers can bookmark different links for different sections of the same cohort. Before sharing a link with families, it is wise to test audio playback on the same Wi-Fi profile students will use, because some school filters block media hosts or large spreadsheet exports.

Dictation workflows that respect classroom pacing

Paper dictation is still valuable for motor memory, but projecting a consistent prompt reduces disputes about what was read aloud. The dictation tool supports per-item timers, optional repetition intervals, and mixed visibility of characters versus Zhuyin so a single template can produce both an on-screen quiz and a printable worksheet. Letter-sized PDF output is convenient for North American schools that stock that paper size by default. Teachers should still preview each export after changing the sheet, because font substitution can affect rare characters or variant glyphs.

Privacy and what this portal stores

This landing page includes a concise Chinese summary and a standalone privacy policy that explains cookies and third-party services such as GitHub Pages hosting and Google Sheets when you paste spreadsheet links. If you enable Google Analytics, place your GA4 measurement ID in the configuration snippet documented in the page source so traffic is measured responsibly. Student-facing pages should never embed hidden trackers that contradict your district policy. When in doubt, share only the tool URLs that load public sheet data and avoid columns that contain personally identifiable information.

Limitations and support expectations

The maintainer updates these tools as a personal teaching project. There is no paid support desk. Response time for email feedback depends on availability. Feature requests are welcome, especially when they improve accessibility or clarify instructions for non-Chinese-speaking homeroom partners. You can also follow or fork the LearnChinese repository on GitHub to inspect HTML changes or suggest edits through pull requests if you are comfortable with version control.

How to cite or share responsibly

When you blog about classroom results, please link both this portal and the specific tool URLs you used so readers can reproduce your setup. If you translate any instructions into another language, note that machine translation may misinterpret pedagogical vocabulary; attaching the original Traditional Chinese wording helps bilingual colleagues verify accuracy. This extended English article is intentionally published on-page so readers can evaluate purpose, audience, and data handling without guessing from a thin marketing blurb alone.

Thank you for teaching Chinese. Whether you serve five students or fifty, structured listening and writing practice still matters. These utilities aim to remove repetitive file prep so you can spend more minutes on corrective feedback, cultural context, and the joyful parts of language class.

自製遊戲內容

聽音辨字的試算表範本（請先開啟後建立副本再編輯）：

開啟 Google 試算表範本

  1. 在範本中按「檔案 → 建立副本」，複製到自己的雲端硬碟。
  2. 依試算表內說明，在指定欄位填入語詞、圖片網址、英文翻譯等。
  3. 按「共用」，將一般存取權設為知道連結的任何人可檢視。
  4. 複製瀏覽器網址列的完整網址，貼在下方，按「生成專屬遊戲網址」。

若網址中沒有 gid，會使用第一個工作表。產生的連結會開啟線上的聽音辨字遊戲頁。

自製語詞考試

聽寫語詞考試試算表範本（請先開啟後建立副本再編輯）：

開啟 Google 試算表範本

  1. 在範本中按「檔案 → 建立副本」，複製到自己的雲端硬碟。
  2. 依試算表內說明編輯：第一列 A 欄須為「目錄」，B～K 欄為第 1～10 題；每格可貼上整段語詞（最多四個中文字）。
  3. 按「共用」，將一般存取權設為知道連結的任何人可檢視。
  4. 複製瀏覽器網址列的完整網址（請保留 gid=… 以指定工作表分頁），貼在下方，按「生成專屬考試網址」。

若網址中沒有 gid，會匯出試算表的第一個工作表。產生的連結會開啟線上的聽寫語詞考試頁。
